# CETACH No. 2 - Web de listas de utiles escolares

Este proyecto incluye una pagina web responsiva y 5 archivos PDF listos para descarga por grado.

## Estructura
- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`
- `assets/hero.svg`
- `pdfs/1ro_primaria.pdf`
- `pdfs/2do_primaria.pdf`
- `pdfs/3ro_primaria.pdf`
- `pdfs/4to_primaria.pdf`
- `pdfs/5to_primaria.pdf`

## Como abrirlo en Visual Studio Code
1. Descarga y descomprime la carpeta.
2. Abre **Visual Studio Code**.
3. Ve a **File > Open Folder** y selecciona la carpeta `cetach2_web`.
4. Abre el archivo `index.html`.
5. Instala la extension **Live Server** si quieres verlo como sitio web local.
6. Haz clic derecho en `index.html` y luego en **Open with Live Server**.
7. Prueba los botones de cada grado. Deben descargar el PDF correspondiente.

## Como subirlo a GitHub
1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos manteniendo la misma estructura de carpetas.
3. Si activas **GitHub Pages**, usa la rama principal y la carpeta root.
4. La web funcionara porque usa rutas relativas locales.

## Personalizacion rapida
- Cambia textos en `index.html`.
- Cambia colores en `styles.css`.
- Si quieres modificar las listas, sustituye los archivos PDF dentro de `pdfs/`.
